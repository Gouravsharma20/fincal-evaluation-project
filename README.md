## Quick start

1. Clone the repository:

```bash
git clone <repo-url>
cd <repo-folder>
```

2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Create a `.env` file (recommended) and add any required environment variables.

4. Start the development server:

```bash
npm start
# or
yarn start
```

5. Sign in using the admin credentials above (local/dev only).
